Hi，我是张振衣，从 2015 年开始从事前端开发工作，现在供职于[心知科技](https://www.seniverse.com)，围绕着天气数据进行商业化运作。

我很喜欢 Open Source 的氛围，曾经参与 [antd](https://github.com/ant-design/ant-design) 和 [@types/react](https://github.com/DefinitelyTyped/DefinitelyTyped/tree/master/types/react) 的研发和评审相关工作。

曾在百度从事代码，云原生，AI服务相关工作。

### 目前维护着以下一些库

- [region-react](https://github.com/regionjs/region) - 状态管理库，支持 react 和 node
- [@hero-u](https://github.com/dancerphil/hero-u) - 一系列工具库，专注于零依赖渐进增强常见的库

  - [@hero-u/hooks](https://www.npmjs.com/package/@hero-u/hooks) - react hooks
  - [@hero-u/css](https://www.npmjs.com/package/@hero-u/css) - @emotion/css 下的常见 className，目前不推荐，更推荐使用 unocss
- [axios-interface](https://github.com/dancerphil/axios-interface) - axios 的封装，注重于接口的定义，另有 [iframe-interface](https://github.com/dancerphil/iframe-interface)

- [use-short-key](https://k83fb.csb.app/) - 一个基于 Keyboard Event 的快捷键注册方案
- [@panda-design](https://panda-design-team.github.io) - 一套基于 Ant Design 演变而来的设计规范，包括：
  - [@panda-design/components](https://www.npmjs.com/package/@panda-design/components) - 组件库
  - [@panda-design/router](https://www.npmjs.com/package/@panda-design/router) - 路由相关的组件解决方案：面包屑、侧边栏
  - [@panda-design/path-form](https://www.npmjs.com/package/@panda-design/path-form) - 大型表单解决方案
- [@koopa-cli](https://github.com/dancerphil/koopa-cli) - 一些 cli 工具
